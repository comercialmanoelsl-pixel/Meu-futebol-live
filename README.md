# DhuniorStats V30 Dados Reais

Incluído:
- Escalações reais somente quando a API entrega lineups.
- Sem escalação simulada.
- Página de time básica usando endpoint /api/team-profile.
- Página de jogadores básica usando dados reais quando a API entrega.
- Tabela real da competição via leagueId/season.
- Timeline com eventos reais.
- Momentum honesto: só aparece quando há estatísticas ao vivo.
- Mensagens claras quando o dado não está disponível.

Build:
npm install

Start:
npm start
