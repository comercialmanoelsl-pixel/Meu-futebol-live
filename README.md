# DhuniorStats V14 — Estatísticas Sempre Úteis

O que mudou:
- As abas laterais agora mudam o conteúdo central.
- Finalizações, chutes no gol, posse e escanteios sempre aparecem.
- Quando a API entrega dados ao vivo, aparece como “Dado ao vivo”.
- Quando a API não entrega, aparece como “Estimativa contextual”.
- Filtro “Com cobertura API” para priorizar jogos com fixtureId.
- Tema claro corrigido, sem fundo preto estranho.
- Sem mensagens técnicas de chaves/API para usuário final.

Render:
Build Command: npm install
Start Command: npm start

Environment obrigatório:
API_FOOTBALL_KEY = sua chave API-Football

Opcional:
NEWS_API_KEY = sua chave NewsAPI
