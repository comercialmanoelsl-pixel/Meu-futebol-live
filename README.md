# DhuniorStats V21 — Layout estilo SofaScore/Flashscore

Mantém backend e APIs, mas troca o frontend para uma experiência mais parecida com sites esportivos profissionais:
- topo fixo com busca
- lista de partidas em cards compactos
- página da partida com placar e abas
- estatísticas em linhas limpas
- painel lateral com IA, eventos e base
- tema claro e tema escuro fixos

Variáveis no Render:
SPORTMONKS_KEY
API_FOOTBALL_KEY
NEWS_API_KEY
OPENWEATHER_KEY
GEMINI_API_KEY

Build:
npm install

Start:
npm start
