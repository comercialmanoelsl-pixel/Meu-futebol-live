# DhuniorStats V31 Estável Profissional

Esta versão prioriza estabilidade, credibilidade e dados reais.

Inclui:
- Mantém a home inicial, sem abrir direto em uma partida.
- Remove heatmap/odds fake quando não há dados reais.
- Escalações só aparecem quando a API entregar lineups reais.
- Mensagens honestas quando a informação não está disponível.
- Tabela real por leagueId/season quando a API entregar.
- Endpoint de perfil do time para últimos/próximos jogos.
- Cache desativado para evitar placar travado.
- Barra de ligas com scroll melhorado.
- Backend preparado para buscar lineups em SportMonks.

Build:
npm install

Start:
npm start
