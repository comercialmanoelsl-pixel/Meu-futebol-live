# DhuniorStats V24 Pro

Inclui em uma versão:
- layout estilo SofaScore/Flashscore com lista lateral fixa
- página da partida com subabas
- Times, Jogadores, Escalações, Tabela, Momentum, IA, Pós-jogo e Notícias
- favoritos por time
- filtros rápidos por liga
- endpoints de tabela e perfil do time
- tema claro/escuro fixos

Variáveis:
SPORTMONKS_KEY
API_FOOTBALL_KEY
NEWS_API_KEY
OPENWEATHER_KEY
GEMINI_API_KEY

Build: npm install
Start: npm start
